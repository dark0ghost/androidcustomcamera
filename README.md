## about
In this project a camera is implemented using the library **jetpack compose**. The application also implements a tsp server that can send photos by sending bits. The project is fully implemented at Kotlin

## resource consumption 
ram - 100-120 Mb<br>
app size - 9 Mb

## Android support 
min Android 9 Pie (API level 29);<br>
max Android 11 (API level 30);

## server command
|name|about|use|
|-----|---|---|
|send | send photo how byte array| send\n|
|set_focus| set focus distance| set_focus\n100f\n |
|set_size_photo| set size photo|set_size_photo\n3840:2100\n|
|get_focus_data| get minimal and maximum focus size| freeze|
